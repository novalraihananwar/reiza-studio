Sistem Keamanan

Proteksi Karya Original

Semua artwork OG dilindungi dari pengambilan paksa melalui sistem multi-layer yang berjalan di browser. Tidak ada tombol download.

🖱️
Right-Click Block
Klik kanan di area gambar langsung dicegah. Gambar langsung blur hitam selama 0.9 detik sebagai peringatan visual.
🚫
Drag Block
Gambar tidak bisa di-drag ke desktop atau tab lain. Intercepted sebelum browser memproses drag event.
⌨️
Shortcut Block
Ctrl+S, Cmd+S, Ctrl+U diblokir. PrintScreen menyebabkan semua gambar blur gelap seketika.
📵
Mobile Protection
Long-press di iOS/Android tidak memunculkan opsi "Save Image". Touch callout dinonaktifkan.
🖨️
Print Block
Jika halaman dicetak (Ctrl+P), semua gambar muncul hitam + blur. Muncul pesan copyright.
🛡️
Shield Overlay
Setiap gambar ditutup div transparan sehingga tidak bisa diinteraksi langsung. pointer-events:none pada semua img.
👁️
Tab Switch Blur
Saat tab tidak aktif (screen capture tool), semua gambar otomatis blur. Kembali normal saat tab aktif lagi.
🔔
Toast Notifikasi
Setiap percobaan download memunculkan notifikasi:
🔒 Artwork dilindungi · © Rytopen
